1. Objetivo do site
    Criar um espaço digital para colaboração entre profissionais/estudantes.
    Permitir troca de ideias, partilha de recursos e acompanhamento de projetos conjuntos.
2. Estrutura básica (páginas/seções)
    Home: Apresentação breve do projeto e chamada para ação.
    Sobre: Explicação da missão, valores e objetivos da colaboração.
    Projetos: Lista dos projetos ativos, com descrição e status.
    Colaboradores: Perfis dos participantes, com bio curta e áreas de atuação.
    Contato: Formulário simples para mensagens ou inscrição.
3. Funcionalidades essenciais
    Navegação clara e responsiva (menu simples).
    Área para destacar projetos em andamento.
    Links para perfis externos (GitHub, LinkedIn, etc.).
    Formulário de contato funcional (HTML + integração básica).
4. Design e estilo
    Layout limpo e moderno (usar CSS Grid/Flexbox).
    Paleta de cores neutra com um tom de destaque (ex.: azul ou verde).
    Tipografia legível e consistente.
    Favicons e ícones simples para reforçar identidade.
5. Tecnologias sugeridas
    HTML5 e CSS3 (estrutura e estilo).
    JavaScript básico para interatividade (ex.: validação de formulário).
    Git/GitHub para versionamento e colaboração.
6. Próximos passos
    Criar a estrutura inicial em HTML (páginas e navegação).
    Definir paleta de cores e tipografia.
    Implementar CSS Grid para layout responsivo.
    Adicionar conteúdo inicial (textos e imagens).
    Publicar no GitHub Pages ou similar para acesso fácil.
